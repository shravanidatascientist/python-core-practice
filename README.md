# Python Core Practice

This repository contains Jupyter notebook practice files focused on Python core concepts.  
It is designed for hands-on learning and clear understanding through examples.

## Topics Covered
- Python variables  
- Python data types  
- String indexing  
- String slicing  
- Advanced slicing  
- Type casting  

## Tools & Technologies
- Python  
- Jupyter Notebook  

## Purpose
The purpose of this repository is to:
- Practice Python core concepts
- Build a strong foundation for data science and further Python learning
- Maintain organized and reusable practice notes

## How to Use
- Clone or download the repository  
- Open the notebooks using Jupyter Notebook or JupyterLab  
- Run and modify the cells to practice the concepts  

## Notes
This repository will be updated as more Python topics are learned and practiced.
